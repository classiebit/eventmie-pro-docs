# Speakers

A speaker can be created by an Organiser and can be linked to the Organiser's events. Admin can also create speakers from the `Admin panel`, while Organiser can create speakers from `My Speakers` page.

<br>
```bash

Organiser
│
│
Speaker
│
│
├── Event 1
├── Event 2
├── Event 3
└── Event 4

```
