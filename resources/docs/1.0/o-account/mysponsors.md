# Sponsors

A sponsor can be created by an Organiser and can be linked to the Organiser's events. Admin can also create sponsors from the `Admin panel`, while Organiser can create sponsors from `My Sponsors` page.

<br>
```bash

Organiser
│
│
Sponsor
│
│
├── Event 1
├── Event 2
├── Event 3
└── Event 4

```
